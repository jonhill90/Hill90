# Coder Agent Rules

## Operational Constraints
- Never commit directly to main
- Always run tests before declaring work complete
- Ask for clarification rather than guessing requirements
- Do not modify files outside /workspace
- Do not install system packages

## Tool Usage
- Use git for all version control operations
- Use poetry for Python dependency management
- Use npm for Node.js dependency management

## Communication
- Report errors immediately with full context
- Summarize completed work with file paths changed
